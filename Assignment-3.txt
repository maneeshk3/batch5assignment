Question 6:
Write java program to print factorial of a number using recursion function.
Ans->
public class Factorial {

    public static int fact(int n){
        if(n>=1){
            return n*fact(n-1);
        }
        return 1;
    }
    public static void main(String[] args) {
        int num=6,facto=1;
        facto=fact(num);
        System.out.println("factorial of number "+num+"= "+facto);
    }
}

Question 11:
Write a java method, which will take array of integers as input and will reverse 
the elements in the array.
Ans->
public class ArrayRevers {
    public static void revers(int[] array){
        int first=0;
        int last=array.length-1;

        while (first<last){
            int temp=array[first];
            array[first]=array[last];
            array[last]=temp;
            first++;
            last--;
        }
    }
    public static void main(String[] args) {
        int []arr={1,2,3,4,5};
        revers(arr);
        for(int n:arr){
            System.out.print(n+",");
        }
    }
}
